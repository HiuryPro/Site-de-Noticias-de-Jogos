Sobre o projeto

Projeto Integrador - Sistemas de Informação ...

Ao curso de Sistemas de Inforção, mais precisamente no primeiro período, nos foi proposto um projeto cuja disciplina é chamada de Projeto Integrado à qual tem como finalidade incluir e desenvolver os conhecimentos adiquiridos durante o primeiro semestre letivo do curso. Assim após a formação de grupos e a escolha de um caminho a qual iríamos seguir, o grupo concluiu que seria feito um site com a finalidade de centralizar e apresentar notícias sobre o mundo dos games, com o objetivo de tornar mais precisa e mais fácil a pesquisa sobre determinado game ou determinado assunto dentro do mundo dos games. Sendo assim este site, é um projeto desenvolvido durante as aulas de P.I onde iremos reunir notícias sobre o mundo dos games dentro do E-sports e aplicando todos os conceitos e conhecimentos que adiquirimos ao longo deste semestre, ultilizando HTML e CSS, com princípios de JavaScript e de Bootstrap para melhor apresentar este mesmo projeto! Fiquem a vontade para apreciar nosso trabalho...!
